# 08. Neuro-Resource Allocator

## Concept
A logic-based engine designed to solve the problem of resource distribution. Instead of manual estimation, it uses a weighted percentage algorithm to allocate resources (time, budget, or energy) across various tasks based on their relative importance.

## Innovation
The project introduces **Proportional Logic**. This is a fundamental concept in data science and AI (like Softmax functions or weight distribution), but implemented here using pure Python arithmetic. It shows an early understanding of how to balance complex systems.

## How to Run
1. Run: `python main.py`
2. Define your total resource (e.g., 40 hours a week).
3. Add your subjects/tasks and their weight.
4. Get a mathematically balanced schedule.
